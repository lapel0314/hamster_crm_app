# Golden Hamster CRM 디자인 시스템

## 디자인 컨셉
- 골든햄스터를 메인 마스코트로 사용한다.
- 업무용 CRM이지만 차갑지 않게, 반려동물샵/분양샵 느낌의 따뜻한 UI.
- 버튼, 카드, 입력창은 모두 둥글고 부드럽게 처리한다.
- 색은 크림/골드/브라운 중심. 보조색으로 민트와 연핑크를 제한적으로 사용한다.

## 컬러 토큰

### 기본
- `background`: `#FFF7E8` — 전체 배경
- `surface`: `#FFFFFF` — 카드/패널
- `cream`: `#FFE8B8` — 비활성 버튼, 헤더 배경
- `gold`: `#F7B733` — 주요 버튼, 강조 그래프
- `deepGold`: `#D98C18` — hover/pressed 강조
- `brown`: `#4A2C14` — 주요 텍스트
- `softBrown`: `#8A5A2B` — 라벨/보조 텍스트
- `line`: `#E9CFA1` — 테두리
- `muted`: `#7E6A56` — placeholder/설명

### 보조
- `mint`: `#B7E4C7` — 구매 그래프/보조 상태
- `pink`: `#FFD6D6` — 부드러운 알림/강조
- `input`: `#FFFDF8` — 입력창 배경

## 타이포그래피
- 기본 폰트: Windows 기본 한글 폰트 또는 Pretendard 권장.
- 앱 타이틀: 32~38px, Bold
- 페이지 제목: 30~36px, Bold
- 카드 제목: 20~24px, Bold
- 본문/표: 13~15px, Regular
- 라벨: 13~14px, Medium/Bold

## 레이아웃
- 기본 해상도 기준: 1440 × 960
- 좌측 사이드바 고정.
- 콘텐츠 영역은 카드 기반.
- 사이드바 폭: 약 250px
- 주요 카드 radius: 24~32px
- 버튼 radius: pill 형태, 22~28px
- 입력창 radius: 16~18px

## 공통 컴포넌트

### 사이드바
- 상단: 햄스터 마스코트
- 타이틀: Golden Hamster
- 서브타이틀: 로컬 오프라인 CRM
- 메뉴: 대시보드, 고객등록, 고객DB, 가망고객
- 하단 안내 카드: “혼자 쓰는 PC용 / 로그인 없이 바로 시작”

### 버튼
- Primary: 골드 배경, 흰색 또는 브라운 텍스트
- Secondary: 크림 배경, 브라운 텍스트
- 저장 버튼에는 `🐹` 아이콘 사용 가능
- 삭제 버튼은 과하게 빨갛게 하지 말고 확인 팝업으로 안전 처리

### 입력창
- 배경: `#FFFDF8`
- 테두리: `#E9CFA1`
- 라벨은 입력창 위에 배치
- 메모 입력창은 높게 구성

### 표
- 표 전체를 둥근 카드 안에 배치
- 헤더는 크림 배경
- 행 구분선은 얇은 `line` 색상
- 고객DB/가망고객은 검색/달력 필터를 상단 필터바에 배치

### 그래프
- 월별 분양 추이: 골드 막대
- 월별 구매 추이: 민트 막대
- 둥근 막대 형태 사용
- 수치보다 시각적 흐름을 먼저 보이게 구성

## 페이지별 가이드

### 고객등록
- 큰 입력 카드 중심.
- 오른쪽에 햄스터 이미지 카드 배치.
- 저장 버튼은 크고 명확하게.
- 메모 입력은 하단 넓은 영역.

### 고객DB
- 상단 필터바: 날짜 달력, 검색창, 검색 버튼, 초기화 버튼.
- 표 컬럼: 날짜, 고객명, 성별, 휴대폰번호, 분양, 구매, 매출, 원가, 메모.
- 검색 결과가 없을 때는 햄스터 일러스트와 빈 상태 메시지 표시.

### 가망고객
- 상담날짜 달력, 방문예정날짜 달력, 검색창을 상단에 배치.
- 방문예정일이 가까운 고객은 부드러운 골드/핑크 배지로 표시 가능.
- 표 컬럼: 상담날짜, 방문예정날짜, 고객명, 성별, 휴대폰번호, 분양, 구매, 매출, 원가, 메모.

### 대시보드
- 상단: 월별 분양 추이, 월별 구매 추이 2개 그래프.
- 하단 좌측: 1월~12월 정산현황.
- 하단 우측: 이번달 분양 순위, 구매 순위.
- 순위 카드는 골드/민트 배지를 사용.

## 이미지 사용
- 마스코트 파일: `hamster_crm/assets/golden_hamster_mascot.png`
- 앱 첫 화면, 사이드바, 고객등록 보조 카드에 사용.
- 너무 많이 반복하지 않고 핵심 위치에만 사용한다.
