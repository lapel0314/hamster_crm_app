# 디자인 브리프 - Golden Hamster CRM

## 핵심 방향
- 로컬 오프라인 Windows 데스크톱 앱.
- 로그인 없음.
- 사용자는 개인 PC에서 혼자 사용.
- 디자인이 핵심 경쟁력.
- 컨셉: 골든햄스터, 따뜻함, 아기자기함, 둥근 버튼, 부드러운 카드 UI.

## 무드
- 귀여움: 햄스터 일러스트/아이콘 사용.
- 신뢰감: 매출/원가/고객DB는 표와 카드로 명확하게.
- 가벼움: 과한 업무용 CRM 느낌보다 반려동물샵/분양샵 느낌.

## 페이지별 UI

### 고객등록
- 큰 입력 카드 중심.
- 오른쪽에 햄스터 이미지/일러스트 카드.
- 저장 버튼은 햄스터 아이콘 포함.

### 고객DB
- 상단 필터 바.
- 날짜 필터는 달력형 입력.
- 검색창과 초기화 버튼.
- 표는 둥근 카드 안에 구성.

### 가망고객
- 상담날짜/방문예정날짜 달력 필터 2개.
- 검색 기능.
- 고객DB보다 일정 관리 느낌을 강조.

### 대시보드
- 월별 분양 추이 그래프.
- 월별 구매 추이 그래프.
- 1월~12월 정산 카드.
- 이번달 분양/구매 순위 카드.
