# Hamster CRM

햄스터 분양 업체용 신규 CRM 프로그램 프로젝트 공간입니다.

## 상태
- 2026-05-06: 랭가 지시로 기존 CRM과 별개인 신규 프로그램으로 정정.
- 현재는 요구사항/설계 준비 단계입니다.
- 아직 서버 실행, 빌드, 배포, 코드 생성은 하지 않았습니다.

## 원칙
- 기존 `crm_app`은 수정하지 않습니다.
- 기존 CRM은 참고 자료로만 봅니다.
- 신규 프로그램의 기술 스택과 MVP 범위 확정 후 개발을 시작합니다.

## 후보 기능
- 고객 문의/상담 관리
- 햄스터 개체 관리
- 예약/방문 일정 관리
- 계약/분양 완료 관리
- 사후관리 이력
- 직원 권한 관리
